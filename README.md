## GitHub Actions

本分支已经集成GitHub Actions，在本分支的提交将会自动触发一次编译行为。一般而言，编译时长大约为5分钟左右。

在最上方的Actions选项卡中查看编译结果，请保证编译可以顺利通过。

编译会产生两个包裹：Slide包裹内含所有课程的放映用幻灯片；Handout包裹内含有所有课程的打印用版本。
